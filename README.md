## Setup
```bash
git clone https://github.com/oscell/PickMe_demo.git
cd PickMe_demo
```

```bash
npm install
```

## Usage

### Start Backend
```bash
npm run server
```

### Start React App
```bash
npm start
```

Go to [localhost:3000](http://localhost:3000/)
